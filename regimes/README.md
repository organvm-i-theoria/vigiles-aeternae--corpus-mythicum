# Regime Specifications

Formal regime YAML definitions live in the META hub repo:
`meta-organvm/vigiles-aeternae--agon-cosmogonicum/regimes/`

before they are formalized into the META hub. Each tradition synthesis
includes a regime draft section — those drafts are promoted here before
being finalized as YAMLs in the engine repo.
