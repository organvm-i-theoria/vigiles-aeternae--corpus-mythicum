# Power Taxonomy Schema

Cross-mythology comparative framework for how each tradition structures:

- **Authority** — how power is granted, delegated, earned
- **Enforcement** — how rules are maintained, violations detected
- **Rebellion** — how the governed resist, overthrow, reform
- **Decay** — how systems degrade, corrupt, entropy
- **Renewal** — how systems regenerate, cycle, rebirth
